Exercice 1 : Sélecteurs et spécificité

Sélecteurs CSS

1.1 Sélecteurs

  1. Tous les liens dans la navigation :
.main-nav a
  1. Uniquement le lien actif :
.main-nav a.active
  1. L’article qui a la classe featured :
article.featured
  1. Le premier paragraphe de chaque article :
article p:first-of-type
  1. Les éléments li pairs :
.main-nav li:nth-child(even)

1.2 Spécificité

Ordre croissant : p < .text < p.text < #main < #main .text p

Règles de même spécificité : la dernière règle écrite l’emporte.

Pourquoi éviter !important : cela rend le code plus difficile à maintenir et complique le débogage.

Exercice 2 : Box Model

Dimensions et centrage

2.1 Comprendre le Box Model

.box {
  width: 300px;
  padding: 20px;
  border: 5px solid #333;
  margin: 15px;
}

1. Avec box-sizing: content-box, la largeur totale visible est 350px.

2. Avec box-sizing: border-box, la largeur totale visible est 300px.

Reset CSS universel :

*,
*::before,
*::after {
  box-sizing: border-box;
}

2.2 Centrer un élément

Pour centrer horizontalement un div de 600px :

div {
  width: 600px;
  margin: 0 auto;
}

Pour le centrer horizontalement et verticalement :

body {
  min-height: 100vh;
  display: flex;
  justify-content: center;
  align-items: center;
}

div {
  width: 600px;
}

Exercice 3 : Flexbox

Disposition flexible

3.1 Barre de navigation

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.nav-links {
  display: flex;
  gap: 20px;
  list-style: none;
  margin: 0;
  padding: 0;
}

3.2 Cartes de même hauteur

.cards {
  display: flex;
  gap: 20px;
}

.card {
  display: flex;
  flex-direction: column;
  flex: 1;
  padding: 20px;
  border: 1px solid #ccc;
}

.card-link {
  margin-top: auto;
}

Exercice 4 : CSS Grid

Mise en page en grille

4.1 Grille de cartes responsive

.grid-cards {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 20px;
}

4.2 Layout complet avec Grid Areas

.page-layout {
  display: grid;
  min-height: 100vh;
  grid-template-columns: 250px 1fr;
  grid-template-rows: auto 1fr auto;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

Exercice 5 : Responsive Design

Mobile First 
/* Base mobile */
.container {
  width: 100%;
  padding: 15px;
}

.nav-menu {
  display: flex;
  flex-direction: column;
  gap: 5px;
}

.content {
  display: block;
}

.sidebar {
  display: none;
}

/* Tablette */
@media (min-width: 768px) {
  .nav-menu {
    flex-direction: row;
  }

  .content {
    display: flex;
    gap: 20px;
  }

  .sidebar {
    display: block;
    width: 30%;
  }

  .main-content {
    width: 70%;
  }
}

/* Desktop */
@media (min-width: 1024px) {
  .container {
    max-width: 1200px;
    margin: 0 auto;
  }

  .content {
    display: grid;
    grid-template-columns: 250px 1fr;
    gap: 30px;
  }

  .sidebar {
    display: block;
    width: auto;
  }
}

Exercice 7 : Transitions et animations

Effets visuels

7.1 Bouton animé

button {
  background: #0066cc;
  color: white;
  padding: 12px 20px;
  border: none;
  border-radius: 6px;
  transition: background 0.3s ease, transform 0.3s ease;
}

button:hover {
  background: #004999;
  transform: scale(1.05);
}

7.2 Spinner de chargement

.spinner {
  width: 40px;
  height: 40px;
  border: 4px solid #ddd;
  border-top: 4px solid #0066cc;
  border-radius: 50%;
  animation: spin 1s linear infinite;
}

@keyframes spin {
  from {
    transform: rotate(0deg);
  }
  to {
    transform: rotate(360deg);
  }
}
